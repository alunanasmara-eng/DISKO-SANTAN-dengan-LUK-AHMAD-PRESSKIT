# DISKO-SANTAN-dengan-LUK-AHMAD-PRESSKIT

## About
A professional press kit for Disko Santan × Luk Ahmad — a hybrid oud/DJ live set exploring maqam heritage through electronic synthesis and dub.

## Contents
- **`/indexz/`** — Live press kit website (HTML + assets)
  - `index.html` — Full press kit webpage
  - `/assets/` — Hero images, logos, press shots
  - `/media/` — Audio mixtape (oreganic 2nd july)

- **`/assets/`** — Source files
  - `/images/` — All photos and logos
  - `/documents/` — Press releases, rider files
  - `/media/` — Audio files

## View Live
### Local Development
```bash
cd indexz
python3 -m http.server 8000
```
Then open: `http://localhost:8000/index.html`
